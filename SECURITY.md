# Security Policy

## Kwetsbaarheden melden
Meld beveiligingsproblemen **niet** via publieke GitHub issues.

Stuur een e-mail naar: **edcafferata@icloud.com**

Vermeld:
- Beschrijving van het probleem
- Stappen om het te reproduceren
- Mogelijke impact

Wij streven ernaar binnen 48 uur te reageren.
