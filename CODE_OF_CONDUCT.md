# Code of Conduct

## Onze belofte
Wij streven naar een open en gastvrije omgeving. Deelnemers verbinden zich ertoe een respectvolle ervaring te bieden voor iedereen.

## Onze standaarden
**Positief gedrag:**
- Respectvol en constructief taalgebruik
- Openstaan voor andere standpunten
- Accepteren van constructieve kritiek
- Focus op wat het beste is voor het project

**Onaanvaardbaar gedrag:**
- Pestgedrag of beledigend taalgebruik
- Persoonlijke aanvallen
- Publiceren van privé-informatie zonder toestemming

## Handhaving
Overtredingen kunnen worden gemeld bij edcafferata@icloud.com.
